# BlockBlast AI 解法分析器

自動分析 Block Blast 遊戲局面，判斷三個待放圖形是否存在合法擺放方式。

## 遊戲規則簡介

- 8×8 格子的棋盤
- 每回合有三個圖形需放入棋盤
- 某行或某列全滿（8格）時會自動消除
- 放置圖形後若同時滿足多行多列全滿，會同時消除
- 難度在於有時需要先放入某個圖形、觸發消行，才有空間放入另一個圖形

## 版本規劃

### v1 — Python 桌面 App（目前開發中）

- 使用 Python + Pygame（或其他框架）製作
- 提供點擊式 UI：使用者自行點擊填入目前 8×8 棋盤狀態
- 提供三個 5×5 格子供使用者繪製三個待放圖形（最大圖形為 1×5 直條）
- 求解器分析所有合法擺放順序（含考慮中途消行後才能放入的情況）
- 輸出是否存在合法擺法，並顯示解法步驟
- 打包成可直接雙擊開啟的執行檔，無需終端機或 IDE

### v2 — Flutter 行動 App

- 將 v1 的核心邏輯移植至 Flutter（Dart）
- 支援 iOS / Android
- 提供更流暢的觸控操作體驗

### v3 — 網站版本

- 以網頁形式提供相同功能
- Python 後端（Flask / FastAPI）負責求解
- 前端提供點擊式棋盤輸入介面

### v4 — 截圖自動辨識版本

- 使用 OpenCV 分析手機截圖
- 自動辨識 8×8 棋盤的佔據狀態（以「是否為背景色」判斷，不依賴方塊顏色）
- 自動辨識下方三個待放圖形的形狀
- 支援多種手機解析度（以相對比例定位格子）
- 串接 v3 網站或獨立執行

## 核心求解邏輯

1. 對三個圖形的 6 種放置順序逐一枚舉
2. 對每種順序，遞迴嘗試每個圖形的所有合法位置（約最多 50 種）
3. 每次放置後模擬消行，更新棋盤狀態，再處理下一個圖形
4. 找到任一合法完整擺法即輸出；若所有組合均無解則回報無解

最壞情況搜尋量約 6 × 50³ = 750,000 種，Python 可即時跑完。
